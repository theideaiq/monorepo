## 2024-04-12 - [Hardcoded yellow color tokens]
**Learning:** Developers are hardcoding `#facc15` (e.g. `bg-[#facc15]`, `text-[#facc15]`) instead of using the `--color-brand-yellow` configuration token from the design system.
**Action:** Replace `#facc15` with `brand-yellow` in classes (e.g., `bg-brand-yellow`, `text-brand-yellow`) to ensure consistent branding and easier theme updates.
