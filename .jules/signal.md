## 2024-03-04 - Semantic HTML Navigation
**Learning:** Having multiple `<nav>` elements on a page is valid HTML5 and improves SEO, but for accessibility (a11y), best practice dictates each should have an `aria-label` or `aria-labelledby` to help screen readers distinguish between them.
**Action:** When adding multiple `<nav>` tags (e.g., in a multi-column footer), assign descriptive `aria-label` attributes to each one.
